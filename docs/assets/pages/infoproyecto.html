información del proyecto


Este sitio ha sido desarrollado en dos instancias consecutivas como Proyecto Integrador (Página Web Responsive) y luego como Proyecto Final (Aplicación Web con conexión a base de datos) para y durante la cursada del tramo educativo avanzado denominado 'Desarrollo Full-Stack Java' del Programa Codo a Codo (Ministerio de Educación, Gobierno de la Ciudad de Buenos Aires, Argentina) durante el primer cuatrimestre del año 2024 como requisito tanto para la titulación intermedia 'Front-End Web Developer' como para la titulación final, 'Full-Stack Web Developer con especialización en Back-End mediante Java'.


2024. Buenos Aires. Argentina
